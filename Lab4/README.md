# Orchids Display Application

A beautiful React application for displaying a collection of orchids.

## Features
- Display 16 different orchids
- Container and Presentation component pattern
- Responsive grid layout
- Beautiful UI with modern styling

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will open at http://localhost:3000

## Project Structure
- `src/data/ListOfOrchids.js` - Data for all orchids
- `src/components/OrchidsContainer.js` - Container component (logic)
- `src/components/OrchidsList.js` - Presentation component (UI)
- `src/components/OrchidCard.js` - Individual orchid card component
